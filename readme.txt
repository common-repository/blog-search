== Description ==
This plugin gives Wordpress the ability to search blog with the enter keyword

=== Installation ===
1) Upload the blog-Search to your Wordpress plugins directory.
2) Activate the plugin from your admin interface.
3) Go to Appearance->Widgets and drag the Blog-Search widget to appropiate sidebar.
